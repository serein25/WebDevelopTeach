HTML Basics

Document Structure

Every HTML file follows a nested structure:


<!DOCTYPE html>
<html lang="en">
  <head>
    <!-- metadata, CSS/JS links, title, etc. -->
  </head>
  <body>
    <!-- visible content goes here -->
  </body>
</html>
  • <!DOCTYPE html> – declares the document as HTML5.
  • <html> – root element; the lang attribute sets the page language.
  • <head> – contains metadata, links to CSS/JS, page <title>, etc.
  • <body> – everything inside appears in the browser window.

Head Section

Inside <head>, you declare metadata and load external resources:

  • <title> – text that shows in the browser tab.
  • <meta> – provides info like character set or viewport. For example: 
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • <link> – links to external CSS files: 
    <link rel="stylesheet" href="style.css">
  • <style> – adds internal CSS directly: 
    <style>
  body { background-color: #f0f0f0; }
</style>
  • <script> – embeds or links to JavaScript: 
    <script src="app.js"></script>

Text & Formatting

Common tags to structure and style text:

  • <h1>…<h6> – headings; h1 is largest, h6 is smallest.
  • <p> – paragraph block. Example: 
    <p>This is a paragraph.</p>
  • <strong> – bold text. Example: 
    <strong>Important!</strong>
  • <em> – italicized (emphasis). Example: 
    <em>Emphasized text</em>
  • <span> – inline container for styling or scripting. Example: 
    <span class="highlight">Colored text</span>
  • <div> – block container used for grouping, layout, and styling. Example: 
    <div class="card">
  <p>Card content here.</p>
</div>

Lists

Two types of lists are most common:

  • <ul> – unordered (bulleted) list: 
    <ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
  • <ol> – ordered (numbered) list: 
    <ol>
  <li>First</li>
  <li>Second</li>
</ol>
  • <li> – list item (used inside <ul> or <ol>).

Links & Media

  • <a> – anchor/hyperlink: 
    <a href="https://example.com" target="_blank">Visit Example</a>
  • <img> – image tag (self-closing). Example: 
    <img src="photo.jpg" alt="A descriptive text" width="200">
  • <video> – embed a video: 
    <video controls width="320">
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support video tag.
</video>
  • <audio> – embed audio: 
    <audio controls>
  <source src="sound.mp3" type="audio/mp3">
  Your browser does not support audio element.
</audio>
  • <iframe> – embed another webpage: 
    <iframe src="https://example.com" width="600" height="400"></iframe>

Tables

Use tables for tabular data:

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>30</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>25</td>
  </tr>
</table>
  • <table> – wraps the entire table.
  • <tr> – table row.
  • <th> – header cell (usually bold/centered).
  • <td> – data cell.

Forms & Inputs

Collect user input using forms:

<form action="/submit" method="POST">
  <label for="username">Username:</label>
  <input type="text" id="username" name="username" placeholder="Enter name">

  <label for="email">Email:</label>
  <input type="email" id="email" name="email">

  <label for="bio">Bio:</label>
  <textarea id="bio" name="bio" rows="4" cols="30"></textarea>

  <label for="country">Country:</label>
  <select id="country" name="country">
    <option value="us">United States</option>
    <option value="ca">Canada</option>
    <option value="other">Other</option>
  </select>

  <button type="submit">Submit</button>
</form>
  • <form> – container for form fields; has attributes action and method.
  • <input> – single‐line input (type text, email, number, etc.).
  • <button> – clickable button (can be submit, reset, or button).
  • <textarea> – multiline text input.
  • <label> – labels an input, improves accessibility; the for attribute must match the input’s id.
  • <select> – dropdown list; contains <option> elements.

Layout & Semantic Sections

Use semantic tags to structure your page:

  • <header> – top section of a page or article (often contains logo/navigation).
  • <nav> – navigation links (menus).
  • <main> – main content area (only one per page).
  • <section> – thematic grouping of content with a heading.
  • <article> – independent, self‐contained piece (e.g., blog post).
  • <aside> – sidebar or tangential content (ads, links).
  • <footer> – bottom section (copyright, footnotes).

Example:

<header>
  <h1>My Website</h1>
  <nav>...</nav>
</header>

<main>
  <section>
    <h2>About</h2>
    <p>Some introduction.</p>
  </section>

  <article>
    <h2>Blog Post Title</h2>
    <p>Written content here.</p>
  </article>
</main>

<aside>
  <p>Related links or ads</p>
</aside>

<footer>
  <p>© 2025 My Site</p>
</footer>

CSS Basics

Selectors

CSS selectors choose which HTML elements to style:

  • * – selects all elements. Example: 
    * { box-sizing: border-box; }
  • element – selects all elements of that type. Example: 
    p { color: #333; }
  • .class – selects all elements with that class. Example: 
    .button { padding: 10px; }
  • #id – selects the element with that ID. Example: 
    #header { background: #f8f8f8; }
  • element, element – comma separates multiple selectors. Example: 
    h1, h2, h3 { margin-top: 0; }
  • element element – descendant selector (nested). Example: 
    nav a { text-decoration: none; }
  • element > element – direct child. Example: 
    ul > li { list-style: square; }

Box Model

Every element is a rectangular “box” consisting of:

  • content – where text and images appear.
  • padding – space between content and border.
  • border – outlining around padding/content.
  • margin – space outside the border.

Example:

.card {
  width: 300px;
  padding: 20px;     /* inside spacing */
  border: 1px solid #ccc;
  margin: 10px auto; /* outside spacing and center alignment */
}

box-sizing: by default, width and height do not include padding/border. If you want them included, add:

*, *::before, *::after {
  box-sizing: border-box;
}

Text & Fonts

  • color – text color. Example: 
    p { color: #555; }
  • font-size – size of text. Example: 
    h1 { font-size: 2rem; }
  • font-family – typeface. Example: 
    body { font-family: Arial, sans-serif; }
  • font-weight – how bold. Example: 
    strong { font-weight: bold; }
  • text-align – horizontal alignment. Example: 
    h2 { text-align: center; }
  • text-decoration – underline, line-through, etc. Example: 
    a { text-decoration: none; }
  • line-height – vertical spacing between lines. Example: 
    p { line-height: 1.5; }

Background & Colors

  • background-color – solid color behind content. Example: 
    header { background-color: #4CAF50; }
  • background-image – set a background image. Example: 
    body {
  background-image: url('pattern.png');
  background-size: cover;
  background-position: center;
}
  • background-size – scale the image (cover, contain, px, %). Example above.
  • background-position – where to position the image (e.g., top left, center).
  • background-repeat – repeat behavior (repeat-x, repeat-y, no-repeat). Example: 
    div { background-repeat: no-repeat; }
  • opacity – transparency (0 to 1). Example: 
    .overlay { opacity: 0.5; }

Positioning

Control where elements appear:

  • position: static; – default flow (ignores top/right/bottom/left).
  • position: relative; – positioned relative to its normal position. Example: 
    .box { position: relative; left: 10px; top: 5px; }
  • position: absolute; – positioned relative to the nearest positioned ancestor. Example: 
    .container { position: relative; }
.child {
  position: absolute;
  top: 20px;
  right: 15px;
}
  • position: fixed; – fixed relative to the viewport (always stays in place). Example: 
    .navbar {
  position: fixed;
  top: 0;
  width: 100%;
}
  • z-index – stacking order (higher number is on top). Example: 
    .modal { z-index: 1000; }
  • display – sets how an element is displayed (block, inline, inline-block, none). Example: 
    span { display: inline-block; }
  • visibility – visible or hidden (element still occupies space). Example: 
    .hidden { visibility: hidden; }

Flexbox

Use Flexbox to create flexible, responsive layouts:

.container {
  display: flex; /* enables flex */
  justify-content: center; /* horizontal alignment */
  align-items: center; /* vertical alignment */
  gap: 1rem; /* spacing between flex items */
}
.item {
  flex: 1; /* each item grows equally */
}
  • display: flex; – turns a container into a flexbox parent.
  • justify-content – horizontal distribution (flex-start, center, space-between, etc.).
  • align-items – vertical alignment on the cross‐axis (stretch, center, baseline, etc.).
  • flex-direction – direction of main axis (row, column, row‐reverse, column‐reverse).
  • flex-wrap – wrap items onto multiple lines if needed (nowrap, wrap, wrap‐reverse).

Grid

CSS Grid lets you create two‐dimensional layouts:

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3 equal columns */
  grid-template-rows: auto;
  gap: 10px;
}
.grid-item {
  background-color: #eee;
  padding: 1rem;
}
  • display: grid; – turns a container into a grid parent.
  • grid-template-columns – defines columns (repeat, px, %, fr units).
  • grid-template-rows – defines row sizes.
  • gap – spacing between grid items (row-gap, column-gap).
  • grid-area – assign items to named areas (more advanced).

Animations & Effects

Make elements move or change over time:

  • transition – smooth change of property when its value changes. Example: 
    .button {
  background-color: #008CBA;
  transition: background-color 0.3s ease;
}
.button:hover {
  background-color: #005f7a;
}
  • animation – define keyframe animations. Example: 
    @keyframes fade-in {
  from { opacity: 0; }
  to { opacity: 1; }
}
.fade-in {
  animation: fade-in 2s ease-in-out;
}
  • transform – rotate, scale, skew, or translate elements. Example: 
    .rotate-45 {
  transform: rotate(45deg);
}
  • box-shadow – shadow effect around a box. Example: 
    .card {
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}
  • filter – visual effects such as blur, grayscale, contrast. Example: 
    img.grayscale {
  filter: grayscale(100%);
}
